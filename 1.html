O html é uma linguagem de marcação utilizada para criar páginas http em um navegador, geralmente utilizada em conjunto com algumas linguagens de programação como CSS e Javascript que auxiliam tanto na funcionalidade como na apresentação da página, transformando uma simples tela branca de diversas formas, sejam para criar uma página que armazena um pudim, a criar um site responsivo que integra banco de dados

  • Doctype: define qual é o tipo de documento a ser utilizad (neste caso o html)

  • Html: Demarca o início e fim do documento html assim podendo receber tanto o conteúdo como textos (p, h3 etc) como estilização (tag style)

  • Head: demarca o local onde será colocado as as tags que demarcam: como o documento agirá, que tipo de caracteres ele suporta, o quanto de conteúdo a página usará de tela quando a mesma for aberta entre outras funcionalidades

  • Meta charset: demarca ostipos de caracteres suportados na página onde no caso do utf-8, permite que o usuário use tanto caracteres padrão quanto emojis

  • Title: Delimita o título da página / nome da aba que corresponde ao documento onde no caso deste documento foi colocado "Exercício 1

  • Body: Delimita aonde o corpo(conteúdo) da tela ficará, isso engloba todos os elementos visuais da tela sejam eles textos, parágrafos, títulos,imagens, formas, cores e etc. Tudo aquilo presente dentro da página pode se encaixar no body
