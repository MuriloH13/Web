O que é o protocolo HTTP

    A comunicação HTTP segue o modelo cliente-servidor:

  1. Cliente faz uma requisição HTTP: Exemplo: ao digitarhttps://exemplo.comno navegador, o navegador envia uma requisição para o servidor como no exemplo abaixo
  1. Servidor responde com uma resposta HTTP: O servidor retorna uma resposta com:
  • Um código de status (como 200 OK, 404 Not Found, etc.),
  • Cabeçalhos HTTP,
  • E o conteúdo (HTML, JSON, imagem, etc.)
